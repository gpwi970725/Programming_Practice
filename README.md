### > Python3 IDLE Shortcut key

- 주석 처리 : Alt + 3
- 주석 해제 : Alt + 4
- 탭(오른쪽으로 들여쓰기) : Ctrl + ]
- 탭 해제(왼쪽으로 내어쓰기) : Ctrl + [
